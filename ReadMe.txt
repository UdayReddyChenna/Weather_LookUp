Weather LookUp
This jar is to forecast tomorrows temperature of a city based on zip code

Steps to Run the Jar
1) Download the Jar
2) From Command Line navigate to the JAR location
3) Enter java -jar Weather_LookUp_ByZip.jar to run the Jar file

commnad: java -jar Weather_LookUp_ByZip.jar

API's Used
1) zipcodeapi has been used to get the city name based on Zip code
2) weather.cit.api has been used to forecast tomorrows weather details
